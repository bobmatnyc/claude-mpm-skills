# **Why you should care**

For code readability and naming consistency, below control structure keywords should be in lower cases:

- if
- else
- elseif
- foreach
- for
- do-while
- while
- try
- catch

# **How we detect**

CAST Highlight counts one occurrence each time a control structure keyword uses an upper case.
