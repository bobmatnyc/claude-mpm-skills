# Too many elseif statements increases costs

This code insight searches all the Functions and Methods  having an Elseif statements.

![5362](https://doc.casthighlight.com/wp-content/uploads/2018/02/map-shadow.png)

## **Why you should care**

For the readability reasons else and if should be used instead of ElseIf.  Try to reduce the number of Functions and Methods having elIf statement and  
review the source code in order to replace it with else if.

## **Business Impacts**

*It is recommended to avoid these in order to ensure the code is more readable and cost effective.*
