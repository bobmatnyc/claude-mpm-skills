# **Why you should care**

Group use declarations are used to cut verbiosity when importing multiple entities from a common nameplace. These declarations help make it easier to identify multiple imported entities belong to the same module. It helps improve readability, allow for the code to be less complicated and less error prone as well.

# **Business Impacts**

It is recommended to use Grouping declarations as it helps improve readability and code accessibility by reducing complexity, thereby reducing costs and improving agility.
