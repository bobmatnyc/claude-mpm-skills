# **Having too much Spaghetti Code can be risky**

This code insight counts one violation each time the keywords **break** and **continue** are encountered.

![5362](https://doc.casthighlight.com/wp-content/uploads/2018/02/map-shadow.png)

## **Why you should care**

Avoid unstructured jumps introduced by instructions that interrupt the control flow.
