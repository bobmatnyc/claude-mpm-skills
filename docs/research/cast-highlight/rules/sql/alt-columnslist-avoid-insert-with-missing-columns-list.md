# **Why you should care**

It is recommended to have the list be symmetrical as it allows for the code to be readable and understandable by the user as well allow for increased efficiency and accessibility. It also helps prevent possible bugs and crashes.

# **Business Impacts**

It is advised to give careful attention to the columns list as improper usage of the list can result in increased financial costs, waste of time and resources.
