# **Why you should care**

It is recommended to correctly indent the lines as it allows for the code to be readable and understandable by the user as well allow for increased efficiency and accessibility. It also helps prevent possible bugs and crashes.

# **Business Impacts**

Ensuring that lines are properly indented allows for the code to be more readable and easier to understand on the technical side as well as making it run better overall. Following this practice saves time and cost.
