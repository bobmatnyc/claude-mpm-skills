# **How we detect**

CAST Highlight counts the number of functions/methods found per file. Depending on thresholds based on benchmark analysis of reference applications, CAST Highlight will decrement the capital of points for the scanned file where the code insight occurs.

![5362](https://doc.casthighlight.com/wp-content/uploads/2018/02/map-shadow.png)

## **Why you should care**

For maintainability and code readability purposes, it is a good practice to limit the number of functions/methods per file.

### CAST recommendations

Reduce the number of methods/functions per file.
