# **Why you should care**

The number of files declared in the *FILE-CONTROL* should not be excessive.

# **How we detect**

CAST Highlight counts one occurrence each time the keyword SELECT is encountered in the FILE-CONTROL section. This code insight triggers depending on thresholds observed in our benchmark repository.
